# Prompt Factory

Algorithm-based prompt optimizer (No LLM, No AI, Just JS alogrithm)
Prompt Factory is a simple AI prompt optimizer that transforms raw prompts into structured and effective prompts for LLM systems.

## Features

- Prompt optimization
- Multiple modes (Professional, Creative, Academic, Business)
- Copy to clipboard
- Minimal industrial UI
- Example prompts

## Technologies

- HTML
- CSS
- JavaScript
